Market Mood with OMI

How to Use it

  • Say the Company Name of the stock to Omi. For example: Analyse the Apple stock for me
  • This will create memory trigger and calls the Stock Sentiment Analyzer app and fetches the sentiment of the stock

Key Features

  • Natural Language Processing for stock intent detection
  • Integration with OpenAI for improved text analysis
  • Real-time stock data fetching from Alpha Vantage
  • Cooldown mechanism to prevent excessive API calls
  • Error handling and logging for robust operation

How It Works

  1. Receive webhook data with stock-related information
  2. Extract stock symbol using natural language processing
  3. Validate the extracted stock symbol
  4. Fetch sentiment data for the valid stock
  5. Analyze and summarize the sentiment data
  6. Return the results as a JSON response
